Se requiere un sistema eléctrico óptimo para una economía 100 % digital

El economista y especialista financiero, Aaron Olmos, explicó que para una economía pase a ser digital, se deben contar con varios recursos para que sea efectiva. “Va más allá de las plataformas digitales”, dijo en entrevista telefónica a Unión Radio este lunes 4 de enero. El tema deriva por las recientes declaraciones de Nicolás Maduro, quien prometió que la economía venezolana 100% digital para este año 2021.

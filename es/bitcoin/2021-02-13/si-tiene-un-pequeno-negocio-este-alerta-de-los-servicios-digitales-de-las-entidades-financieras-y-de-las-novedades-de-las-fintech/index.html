Si tiene un pequeño negocio esté alerta de los servicios digitales de las entidades financieras ( y de las novedades de las fintech )

La entidad ofrece líneas de crédito diseñadas en alianza con el Sistema de Banca para el Desarrollo para pymes. Las pequeñas empresas pueden utilizar diversidad de servicios completamente en línea para obtener financiamiento para capital de trabajo. (Imagen archivo) (Shutterstock para EF). La Mutual Cartago de Ahorro y Préstamo (Mucap) puso a disposición una aplicación móvil denominada Pymes Digital para que las pequeñas y medianas empresas soliciten financiamiento, realicen la gestión completa del proceso de préstamo, puedan darle seguimiento a su solicitud y, finalmente, formalizarlo desde cualquier dispositivo electrónico.

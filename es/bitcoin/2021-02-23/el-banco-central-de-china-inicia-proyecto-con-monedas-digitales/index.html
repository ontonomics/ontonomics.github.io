El Banco Central de China inicia proyecto con monedas digitales

El Banco Popular de China tiene su propio proyecto nacional de CBDC, el e-CNY, una de las iniciativas más avanzadas de este tipo en el mundo. El proyecto tiene como objetivo desarrollar un proceso para pagos transfronterizos de divisas en tiempo real bajo tecnología de contabilidad distribuida, que sustenta las criptomonedas como el bitcóin.

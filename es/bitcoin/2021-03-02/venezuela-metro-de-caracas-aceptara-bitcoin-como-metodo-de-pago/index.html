Venezuela : Metro de Caracas aceptará Bitcoin como método de pago

El Metro de Caracas comenzará a aceptar pagos en Bitcoin (BTC), como parte de su plan para migrar su sistema de efectivo a lo digital. Este método de pago se realizará gracias a una tarjeta automatizada. La cual ha comenzado a venderse a partir de este 1ro de marzo.

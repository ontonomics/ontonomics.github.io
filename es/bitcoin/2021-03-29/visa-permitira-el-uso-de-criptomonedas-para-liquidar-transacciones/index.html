Visa permitirá el uso de criptomonedas para liquidar transacciones

Esta alianza marca un paso importante en la estrategia de red de la compañía, que está diseñada para mejorar todas las formas de movimiento de dinero. El primer banco de activos digitales autorizado por el gobierno federal de Estados Unidos, se puso en marcha el proyecto piloto que permite a Crypto enviar USDC a Visa para liquidar una parte de sus obligaciones para el programa de tarjetas Visa.

Blockpit aterriza en España con un software para simplificar la declaración fiscal de criptomonedas

El software, que ya se puede usar en España, Francia, Austria, Alemania, Suiza y Estados Unidos, permite importar datos de transacciones de las principales casas de cambio de criptomonedas (exchange) y monederos personales ('wallets') y los analiza de forma automática. El usuario puede obtener su informe fiscal con toda la información necesaria para la Agencia Tributaria.

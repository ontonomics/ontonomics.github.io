NFT o token no fungible : código fuente de la World Wide Web sale a la venta

El código fuente de Tim Berners-Lee para la World Wide Web es el último token no fungible (NFT) que sale a la venta. La casa de subastas Sotheby’s de Nueva York vende el programa que abrió el camino a Internet que hoy conocemos más de 30 años después de su creación.

Siete mecanismos que se usan para robar datos personales y bancarios

Este mecanismo, conocido como ‘phishing’, tiene como propósito engañar a las personas para que proporcionen contraseñas, números de tarjetas de crédito y otra información confidencial. Para lograr este objetivo se presenta al usuario páginas adulteradas de instituciones financieras, empresas y otras. El fin es acceder a los recursos bancarios de las víctimas o lograr que la gente realice depósitos para garantizar la supuesta entrega del premio ofrecido.

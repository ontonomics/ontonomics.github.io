Huobi Primelist cotizará el token RIFI para permitir los préstamos abiertos seguros

Con un fuerte equipo técnico, una gran base de usuarios y recursos del ecosistema, Huobi puede proporcionar un apoyo sustancial a Rikkei Finance, la mayor bolsa de activos digitales del mundo. El nuevo proyecto ofrece acceso preferente al nuevo plan DeFi, que permite realizar préstamos abiertos de forma segura.

Banco suizo lanza token de oro regulado

SESEBA Bank, una plataforma bancaria de activos digitales con licencia FINMA totalmente integrada, anunció el lanzamiento de su Gold Token, un token digital regulado para la inversión y la entrega de oro físico bajo demanda. El sector crypto busca un marco regulatorio para capitalizar las oportunidades.

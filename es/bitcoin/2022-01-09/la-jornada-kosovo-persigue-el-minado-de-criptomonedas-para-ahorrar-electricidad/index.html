La Jornada - Kosovo persigue el minado de criptomonedas para ahorrar electricidad

La policía de Kosovo incautó el sábado de centenares de dispositivos para minar criptomonedas y detuvo a una persona, pues debido a la actual crisis energética el gobierno prohibió temporalmente esa actividad. Para crear (minar) criptomonedas como el bitcóin, hay que resolver ecuaciones complejas con unos ordenadores muy potentes, y esto consume una enorme cantidad de energía.

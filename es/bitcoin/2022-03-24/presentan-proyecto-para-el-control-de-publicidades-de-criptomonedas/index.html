Presentan proyecto para el control de publicidades de criptomonedas

El senador Andrada se reunió con el vicepresidente de la CNV, Sebastián Negri, para dialogar sobre la instrumentación de mecanismos para “proteger a la ciudadanía de caer en posibles estafas vinculadas con criptoactivos producto de publicidad engañosaografía.org”. El proyecto ha sido aprobado por el Congreso de los Diputados.

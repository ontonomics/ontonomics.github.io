Tailandia prohíbe usar criptomonedas para efectuar pagos

El regulador de valores de Tailandia anunció que prohibirá el uso de criptomonedas para el pago de bienes y servicios, como parte de una medida que protege a la seguridad de los sistemas financieros de ese país de amenazas cibernéticos.

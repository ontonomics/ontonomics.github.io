Santander y Oxentia Foundation lanzan reto global sobre tecnologas blockchain para emprendimientos | Notas

El objetivo de ambas instituciones es apoyar la innovacin para mejorar la productividad y el empleo, as como resolver problemas relevantes para la sociedad. Este nuevo reto global est dirigido a startups y scaleups de 11 pases (Alemania, Argentina, Brasil, Chile, EEUU, Espaa, Mxico, Portugal, Polonia, Reino Unido y Uruguay), que presenten un producto innovador y de base tecnolgica.

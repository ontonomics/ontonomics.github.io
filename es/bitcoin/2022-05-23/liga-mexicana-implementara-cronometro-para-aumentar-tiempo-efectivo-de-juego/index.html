Liga mexicana implementará cronómetro para aumentar tiempo efectivo de juego

La Liga del fútbol mexicano implementará un cronómetro a partir del torneo Clausura 2023, que se iniciará el próximo enero, para aumentar el tiempo efectivo de juego, informó este lunes el presidente del campeonato, Mikel Arriola. El jerarca señaló que en julio se harán pruebas para determinar cuál es el máximo ideal para llevar a cabo las reanudaciones.

Hoy Tamaulipas - Pensin para adultos mayores BIENESTAR Arrancan inscripciones para la Pensin para el Bienestar

La Secretaría de Bienestar puso en marcha este lunes la nueva etapa de registro para solicitar la incorporación a la Pensión para el Bienestar de las Personas Adultas Mayores, para quienes cumplen 65 años entre los meses de julio y agosto. El registro estará vigente en todo el país hasta el 15 de agosto próximo.

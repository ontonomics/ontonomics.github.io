WEMADE se asocia con Code States para inaugurar una academia de cadena de bloques

SESEÚL, Corea del Sur, 2 de noviembre de 2022 /PRNewswire/ -- El 31 de octubre, WEMADE anunció su plan para inaugurar una academia de cadena de bloques con Code States, una empresa emergente dedicada a promover el talento en TI. El programa incluye tres horas de conferencias en línea en tiempo real durante las noches de días de semana.

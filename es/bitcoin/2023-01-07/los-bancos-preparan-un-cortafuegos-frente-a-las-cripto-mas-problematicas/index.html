Los bancos preparan un cortafuegos frente a las cripto ms problemticas

Los bancos europeos se preparan para marcar distancias en el 2023 frente a las monedas virtuales ms problemticas, entre ellas el bitcoin o el ethereum. No ocurrir lo mismo con las que se basan en activos reales, las conocidas como stablecoin. A partir de este ao, los bancos activarn sus cortafuegos para dejar el menor rastro posible de criptomonedas problemticas en sus balances.

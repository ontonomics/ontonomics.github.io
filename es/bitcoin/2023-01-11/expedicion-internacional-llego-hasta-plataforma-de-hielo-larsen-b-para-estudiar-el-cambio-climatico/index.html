EXPEDICIÓN INTERNACIONAL LLEGÓ HASTA PLATAFORMA DE HIELO LARSEN B PARA ESTUDIAR EL CAMBIO CLIMÁTICO

Un grupo de científicos navegó por el mar de Weddell para realizar estudios sobre microorganismos, líquenes, musgos, plantas, algas, invertebrados marinos, peces, junto con pingüinos Adelia y emperador. El resultado fue el resultado del trabajo entre SEDNA Expeditions y el Instituto Milenio BASE para la campaña antártica 2022-2023.

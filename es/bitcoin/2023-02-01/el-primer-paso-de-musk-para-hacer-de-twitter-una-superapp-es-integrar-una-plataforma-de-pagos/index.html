El primer paso de Musk para hacer de Twitter una superapp es integrar una plataforma de pagos

Elon Musk quiere que la red social sea mucho más que una plataforma de microblogging, una forma en la que los usuarios se expresan a base de comentarios cortos. El magnate pretende que sirva para otras muchas cosas, que sea una "superaplicación" que capte la atención y el tiempo del usuario durante la mayor parte del día. Y para lograrlo necesita que se puedan hacer transacciones económicas.

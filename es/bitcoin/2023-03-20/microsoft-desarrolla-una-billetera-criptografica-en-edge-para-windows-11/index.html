Microsoft desarrolla una billetera criptográfica en Edge para Windows 11

La billetera criptográfica de Edge parece estar bien pensada y diseñada para ofrecer una interfaz agradable y fácil de usar. Entre las características se encuentra la capacidad de ver los activos, incluidos los fondos criptográficos, las tendencias de precios, así como la posibilidad de enviar o recibir criptografía utilizando direcciones, nombres o nombres ENS conocidos.
